# 数据结构与算法分析 & C++高级特性的研究

### 目标

1. 充分理解各数据结构的底层运行逻辑。
2. 深度掌握并使用 C++ 的高级特性（面向对象、内存管理、模板编程等）。
3. 最终自主实现一个高质量的数据结构库（具备 STL 风格接口与工程级组织结构）。

---

## **第一阶段：C++核心基础**

### 1. 面向对象编程
* 类与对象：封装、继承、多态、抽象
* 构造函数与析构函数，和相应的深拷贝、浅拷贝问题
* 运算符重载（拷贝赋值、比较、流输出等）
* 友元函数与友元类
* 静态成员与常量成员
* **C++对象模型**（this指针、虚函数表、内存布局）

### 2. 内存管理与拷贝控制
* 栈与堆的内存分配
* `new/delete` 与 `malloc/free` 的区别与使用规范
* **RAII（资源获取即初始化）思想**
* 三/五法则（拷贝构造、拷贝赋值、析构、移动构造、移动赋值）和零法则
* 移动语义与右值引用（C++11+）
* 内存泄漏检测与防范（Valgrind / Sanitizer 工具，未实践）

## **第二阶段：数据结构实现**

### 3. 线性结构

#### 动态数组（Vector）
* 动态扩容策略与内存管理
* 模板类实现
* 异常安全保证（强/基本/无异常保证）
* 迭代器实现（随机访问迭代器）
* 拷贝与移动构造函数优化

#### 链表（Linked List）    <---- 当前进度
* 循环链表实现
* 节点与迭代器封装
* 智能指针（`unique_ptr` / `shared_ptr`）在链表中的应用
* 异常安全性与内存释放设计

### 4. 栈与队列
* 基于数组与链表的栈/队列实现
* 模板编程应用
* 适配器模式（将底层容器封装为不同抽象接口）

### 5. 树结构

#### 基础的树结构
- 二叉树
  * 递归与非递归遍历（前中后序、层次）
  * 模板类与节点泛型化设计
- 二叉搜索树（BST）
  * 插入、删除、查找逻辑
  * 异常与边界处理机制

#### 红黑树与set/map
* 红黑树性质与平衡调整
* 插入与删除操作的各种情况分析与实现
* 基于红黑树封装 `set` 与 `map`
* 迭代器实现（中序遍历支持）
* `key_compare` 与 `value_compare`

### 6. 哈希结构
* 哈希表原理与实现
* 冲突解决策略：链地址法、开放定址法
* 模板化键值存储（支持任意类型）
* 自定义哈希函数与 `std::hash` 特化机制
* `unordered_map` / `unordered_set` 风格接口

### 7. 堆与优先队列
* 二叉堆实现（最小堆与最大堆）
* 模板化比较函数（`Compare` 模板参数）
* 堆排序算法
* 模板优先队列封装
